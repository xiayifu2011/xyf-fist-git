# if条件判断

 如要在C++中表达判断 $x \in [L,R]$,千万不能这么写：
  ``` C++
    if(L<=x<=R){
        ...
    }
  ```
 如果写成这样，C++会先判断`L<=x`得出一个bool型的值，在用这个0或1去和`R`比较
 要这么写：
  ``` C++
    if(L<=x&&x<=R){
        ...
    }
  ```

# 浮点变量问题
 能用double,就不用long double或float
 
 能用float,就不用long double

 运行这段代码
 ```C++
 #include<bits/stdc++.h>
 using namespace std;
 float a;
 double b;
 long double c;
 int main(){
     a=b=c=3.14159265358;
     printf("%.10lf\n%.10lf\n%.10lf\n",a,b,c);
     return 0;
 }
 ```
 结果如下
$$
   \begin{align*}
   & 3.1415927410\\
   & 3.1415926536\\
   & 0.0000000000\\
   \end{align*}
$$

### if条件判断

 如要在C++中表达判断 $x \in [L,R]$,千万不能这么写：
  ``` C++
    if(L<=x<=R){
        ...
    }
  ```
 如果写成这样，C++会先判断`L<=x`得出一个bool型的值，在用这个0或1去和`R`比较
 要这么写：
  ``` C++
    if(L<=x&&x<=R){
        ...
    }
  ```

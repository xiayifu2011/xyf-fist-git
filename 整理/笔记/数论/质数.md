# First
  质数的定义：
   对于 $ p>1 $ 且 $ p\in\mathbb{N}$ 、
   若  $p$ 满足有且仅有 $\pm 1$ 与 $\pm p$ 两个因数
   则称 $p$ 为质数
# 质数判断
 ## 单个质数判断
  发现:若$a \mid b$则$\frac{b}{a}\in \mathbb{Z} $
  故枚举$[1,\sqrt n]$内所有数即可,复杂度 $O(\sqrt n)$
  ```C++
    for(int i=1;i*i<=n;i++){
      if(n%i){
        return 0;
      }
    }
    return 1;
  ```
